# Pre-Installation Checklist

- [ ] Hardware test passes
- [ ] Sensor level clearly changes with light
- [ ] Alarm threshold is calibrated
- [ ] Red LED mirrors the boiler LED
- [ ] Buzzer and mute button work
- [ ] Home Assistant integration is online
- [ ] Last alarm timestamp updates
- [ ] Temperature sensor reports a plausible value
- [ ] Enclosure and sensor holder fit correctly
