# Felsökning

## Röd LED lyser alltid

- Tröskeln är för låg.
- Sensorn får omgivningsljus.
- Fototransistorn kan vara felvänd.

## Röd LED reagerar inte

- Tröskeln är för hög.
- IO34 är felkopplad.
- Sensor saknar 3V3/GND.
- Fototransistorn kan vara felvänd.

## Summern låter inte

- Kontrollera 2N7000-pinout.
- Kontrollera att Buzzer Enabled är på.
- Kör Test Buzzer.
