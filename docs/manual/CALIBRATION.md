# Calibration

1. Read `Boiler Monitor LED Level` while the boiler LED is off.
2. Read it again while the boiler LED is on.
3. Set `LED Threshold` approximately halfway between the two readings.
4. Confirm `Boiler LED Detected` only changes to ON when the boiler alarm LED is lit.
5. Shield the sensor from ambient light using the printed holder or black heat-shrink tubing.
