# Enclosure and Sensor Holder

Included models:

- `boiler_alert_case.scad` / `.stl`
- `boiler_monitor_case.scad` / `.stl`
- `sensor_holder.scad` / `.stl`

Recommended print settings:

- Black PETG or PLA
- 0.2 mm layer height
- 20-40% infill
- Print the sensor holder in black to reduce ambient-light leakage
