# Build Guide

1. Flash `firmware/release/hardware-test.yaml`.
2. Verify the green LED, red LED, buzzer, and button.
3. Build the phototransistor sensor circuit.
4. Add the DS18B20 if required.
5. Flash `firmware/release/boiler-monitor.yaml`.
6. Add the ESPHome device to Home Assistant.
7. Calibrate the LED threshold.
8. Print and install the enclosure and sensor holder.
