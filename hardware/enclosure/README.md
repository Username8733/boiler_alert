# Kapsling och sensorhållare

OpenSCAD-filerna är konceptmodeller:

- `boiler_monitor_case.scad` – enkel kapsling med hål för LED, summer och knapp
- `sensor_holder.scad` – svart sensorhållare/tub för fototransistorn

Exportera STL från OpenSCAD:

1. Öppna `.scad`-filen.
2. Tryck F6 för render.
3. Exportera som STL.
4. Skriv ut i PETG eller PLA. Sensorhållaren bör helst skrivas ut i svart material.
