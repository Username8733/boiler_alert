# Files Removed in v2.0.0-rc4

The following categories were intentionally removed to simplify the repository:

- `.github/workflows/`
- `.github/ISSUE_TEMPLATE/`
- `.github/pull_request_template.md`
- `.github/dependabot.yml`
- `CODE_OF_CONDUCT.md`
- `CONTRIBUTING.md`
- `SECURITY.md`
- `ROADMAP.md`
- `release/`
- `firmware/source/`
- `examples/`
- Individual PDF files under `images/functions/`
- `docs/manual/FUNCTION_DIAGRAMS.pdf`
- Separate component Markdown files that duplicated the diagrams
- Internal language-audit and checksum reports

These files can be reintroduced later if the project grows or begins accepting external contributions.
