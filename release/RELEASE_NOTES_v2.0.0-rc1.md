# ESPHome Boiler Monitor v2.0.0-rc1

This release candidate is intended for content and installation review before v2.0.0.

## Included

- Single-file production firmware
- Single-file hardware-test firmware
- English Home Assistant dashboards and notification example
- English function diagrams in PNG and PDF
- Combined printable diagram manual
- Bill of materials and connection table
- Enclosure and sensor-holder STL/SCAD files
- Detailed README and full project history
- Installation, calibration, diagnostics, enclosure, and troubleshooting documentation
- GitHub project templates and automation
